A miniscule Javascript library for making custom CSS rules

CSS Rules..?

What is a CSS Rule? A CSS Rule is just a selector. Here are some examples of selectors: 

h2
#navbar
.animated

All of Ruleify

rule.get( 'ruleName' ) //returns the rule
rule.add( 'ruleName' ) //adds a CSS rule
rule.kill( 'ruleName' ) //removes a CSS rule
rule.prop ( ruleName, propertyName, property) // adds property to ruleName
// rule.prop('h2', 'font-size', '4em');

Generate CSS rules on-the-fly

rule.add( 'h2' ) // generates a CSS rule: h2 {}

Add classes dynamically

Good for custom animations! 
rule.add( '.animated' );
rule.prop( '.animated', '-webkit-transition', '2s');
rule.prop( '.animated', 'transform-y', '45deg');

if (userWants == '45deg'){ 
    var deg = '45deg'; }
else { var deg = '90deg'; }
    rule.prop( '.animated', 'transform-y', deg);

document.getElementById('element').classList.add = ('.animated');
